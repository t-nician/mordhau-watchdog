# Installation
### Requirements
*   Python 3.10+
*   git

### Instructions
```
git clone https://github.com/t-nician/rcon-watchdog
```

```
cd rcon-watchdog
```

```
copy example.config.jsonc config.jsonc
```

Make edits to config.jsonc to match your rcon server information.

```
pip install -r ./requirements.txt
```

```
python ./src/main.py
```
